# CTF Byte-Cendrars
Voici le CTF Byte-Cendrars. Un jeu de Capture The Flag (Jeopardy) de niveau débutant++ créé dans le cadre d'un travail de maturité au lycée ------- 


## Lancement
Si vous lisez ce README, c'est probablement pour installer le jeu sur un site internet ou pour y jouer en local. 
Si vous souhaitez jouer en local, ne lancez aucun autre fichier qu'index.html pour conserver la difficulté du jeu. 
Une fois index.html lancé, vous pourrez vous balader dans le site. Je vous conseille d'investiguer en premier sur la page de profil accessible en cliquant 
sur la photo de profil en haut à droite. vous y trouverez le site secret de l'élève où vous pourrez entrer les flags que vous avez trouvés.
Pour trouver les épreuves, je ne vous donne aucun conseil, si ce n'est qu'aucun détail n'est laissé à l'abandon. Alors, soyez attentif, explorez chaque détail, et bonne chance !
Les solutions se trouvent uniquement dans l'écrit du travail de maturité.


## Lore
Comme beaucoup de CTF, j'ai décidé de faire une petite mise en scène. 
Dans ce CTF, vous incarnez un enquêteur chargé de découvrir les mots de passe (flags) dissimulés par un élève. 
Votre mission : accéder à son site secret en récupérant ces mots de passe cachés dans différents éléments de sa session.
L'élève en question, Thomas Scholz, a laissé des indices, mais seul un œil attentif pourra percer ses secrets.
